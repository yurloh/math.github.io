- 📈 Play from our **rapidly growing** selection of games
- 🏫 **Unblock websites** with our built in web proxy
- 🫣 **Hide from teachers** using the tab cloak setting

### Installation

```bash
git clone https://github.com/Radon-Games/Radon-Games.git
cd Radon-Games
pnpm install
pnpm run build
pnpm run preview
```
